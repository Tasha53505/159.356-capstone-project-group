MP3 Tag Format

The MP3 tag format does not provide a standard way of defining an Album Artist. Some MP3 tagging tools use the TPE2 field for Album Artist (iTunes, Winamp, Windows Media Player) while others may use it for the intended meaning of "Band/orchestra". Select the meaning you would like Logitech Media Server to use. Changing this setting will start a rescan of your music library.
